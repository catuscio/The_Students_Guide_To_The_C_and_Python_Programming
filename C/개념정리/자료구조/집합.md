# 집합
## 집합 ADT
집합 ADT는 유일한 개체들을 담는 용기를 모델링한다. 이에 따라, 효율을 위해 정렬된 리스트의 형태로 표현한다.

## 메쏘드
- __set union(B)__ : 집합 B와의 '합집합'을 반환
- __set intersect(B)__ : 집합 B와의 '교집합'을 반환
- __set subtract(B)__ : 집합 B를 차감한 '차집합'을 반환
> 집합 A와 B에 관한 주요 작업의 실행시간은 O(|A|+|B|)이어야 함

### 일반 메쏘드
isEmpty, size, elements

### 질의 메쏘드
- __boolean member(e)__ : 집합에서의 e의 유무 반환
- __boolean subset(B)__ : 집합이 집합 B의 부분집합인지 반환

### 갱신 메쏘드
- __addElem(e)__ : 집합에 e 추가
- __removeElem(e)__ : 집합에 e 삭제

### 예외
- __emptySetException()__ : 비어 있는 집합에 대해 삭제 혹은 첫 원소에 접근 시도할 경우 발령

---
## 집합의 이모저모
1. 집합을 연결리스트로 구현할 수 있다
    - 각 노드는 하나의 집합 원소를 표현하고, 이중연결리스트를 주로 사용한다
2. 원소들은 정렬된다
3. 기억장소 사용 - O(n)
4. 배열로 구현할 수도 있다
5. __*합/교/차집합 메쏘드는 파괴적이다. 즉 대상 집합의 원형이 보존되지 않는다.*__
6. 성능
    - 모두 O(|A|+|B|) 시간에 수행된다
    - 단, addLast 작업을 O(1) 시간에 수행하는 경우에만 해당된다
